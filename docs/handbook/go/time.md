# time.Duration

## Что это

`time.Duration` — стандартный тип Go для хранения промежутков времени.

Внутри представляет собой отдельный тип на основе `int64`.

```go
type Duration int64
```

## Почему не int

Использование обычного `int` не позволяет понять, в каких единицах хранится время.

Например:

```go
Duration: 205
```

Неочевидно, что означает число `205`:

- секунды;
- миллисекунды;
- минуты.

`time.Duration` делает смысл значения явным и предоставляет готовые методы для работы со временем.

## Почему мы используем его в проекте

Поле `Duration` модели `Track` хранится как `time.Duration`.

Каждый Provider самостоятельно преобразует длительность, полученную от внешнего API, в `time.Duration`.

Таким образом доменная модель не зависит от формата данных конкретной музыкальной платформы.
